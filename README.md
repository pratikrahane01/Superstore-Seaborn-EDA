# Superstore-Seaborn-EDA
 Superstore Sales Analysis using Seaborn  Hi, I'm Pratik Rahane.  This project is part of my learning journey in data visualization using Python. After completing the Seaborn library, I built this project to apply all major concepts on a real-world dataset and improve my exploratory data analysis skills. 

## plot used 
- Histogram with KDE & Hue  
- Boxplot and Violin plot  
- Bar plot with grouping  
- Jointplot (Hex)  
- Heatmap (Pivot table)  
- FacetGrid  
- Relplot  
- Line plot (Time series)  
- Point plot
